# RESTful API для управления списком задач (ToDo list) с использованием FastAPI и SQLite.

## Возможности
CRUD операции: Полный набор операций для управления задачами

Мягкое удаление: Задачи помечаются как удаленные, а не удаляются физически

Валидация данных: Автоматическая проверка входных данных с помощью Pydantic

Версионирование API: API версии v1 с префиксом `/api/v1`

Автоматическая документация: Интерактивная документация через Swagger UI и ReDoc

## Технологии
FastAPI - современный асинхронный веб-фреймворк

SQLite3 - легковесная встроенная база данных

Pydantic - валидация и сериализация данных

Python 3.7+ - язык программирования

## Установка

Клонируйте репозиторий или скопируйте код

Установите зависимости:

pip install fastapi uvicorn sqlite3

Запустите сервер:

uvicorn main:app --reload

docker 

docker run -d -p 8000:80 -v todo_data:/app/data atatarinov12/todo-service:latest
